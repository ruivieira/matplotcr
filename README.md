# matplotcr

A Crystal wrapper for matplotlib.

# Features

- Line plots
- Scatter plots
- Histograms
- Line segments
- Axis (horizontal and vertical) lines
- Font customisation
- Annotations
- Custom size and DPI
- Sub-plots
- Horizontal and vertical limits
- Custom markers

# Examples

Examples can be found [here](docs/README.md).

# Requirements

`matplotcr` additionally requires:

* Python (3.x)
* matplotlib
* numpy
* scipy (for KDE plots)

to be installed.
